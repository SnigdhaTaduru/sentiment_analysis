Sentiment Analysis

Sentiment Analysis is the process of identifying and categorizing opinions in order to determine whether the writer's attitude towards a particular topic, product, etc. is positive, negative, or neutral.

This project deals with sentiment analysis of data from twitter database.

We use Naive Bayes classification technique to classify tweets.
